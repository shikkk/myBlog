## 常用命令
命令|备注
--|:--:|
status|查看状态
branch|查看分支列表
branch -r|查看远程主机分支
checkout|切换分支
checkout -b|新建本地分支
commit -m|提交代码到主机
log|查看提交记录
merge|合并代码
stash|未修改好的代码储存起来
stash pop|释放储存器的代码
stash list|查看stash列表
diff|代码对比
remote -v|查找远程url
remote set-url origin xxxxxx|替换远程url
fetch|更新远程主机全部取回本地
show-ref|显示所有分支、tags列表
